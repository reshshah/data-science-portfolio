The Framework

Proposed: The Thumbtack Agentic Marketing Operating System (AMOS)

A layered agent stack, owned by the CMO org, built by Applied AI — orchestrating campaigns, personalization, and pro-matching autonomously within human-set guardrails.

Orchestration
A supervisor agent sets goals (CAC target, category mix, local density) and routes work to specialist agents; enforces brand, budget, and compliance guardrails; escalates exceptions to marketers.
Specialist Agents
Campaign Generation · Local-Market Demand Forecasting · Bid & Budget Allocation · Lifecycle/Retention · Creative & Content · Pro-Side Demand Gen (see next slide)
Shared Context
Customer + pro embeddings, real-time GMV/CAC signals, category taxonomy, local-market density data — a feature store all agents read/write to, keeping decisions consistent.
Tools & Actions
Ad platform APIs (Google, Meta, TikTok), Thumbtack CRM/lifecycle system, CMS/creative tooling, pricing & bidding systems — agents act directly, with spend caps and approval thresholds.
Human-in-the-Loop
Marketers set strategy and approve above-threshold actions; weekly agent performance review replaces manual campaign-by-campaign management.

How To Build It

Building AMOS, one agent at a time — a worked example

Rather than build the full stack at once, each agent follows the same build pattern. Here's how it applies to the Local-Market Demand Agent, the logical starting point.

1

Define the decision, not the dashboard

The agent's job is a decision: how much budget should metro X get this week, across paid and organic. Not a report — an action with a clear owner and a reversible blast radius.

2

Wire up the shared context it needs

Pull category demand signals, current CAC/LTV by metro, and pro-supply density into the shared feature store — the same store other agents (Campaign & Creative, Marketing Mix) will read from later.

3

Give it narrow, scoped tool access

Read access to performance data; write access to budget allocation within a capped range (e.g. ±15% of current spend per metro per week) — no unbounded authority on day one.

4

Run it shadow, then supervised, then autonomous

Weeks 1–2: agent recommends, human decides. Weeks 3–6: agent acts within caps, human reviews weekly. Beyond that: autonomous within caps, exceptions escalate automatically.

5

Reuse the pattern for the next agent

Campaign & Creative Agent follows the same five steps, reading from the same shared context — this is what makes AMOS a system rather than five one-off projects.

Why start here

The Local-Market Demand Agent touches budget allocation directly — the fastest path to a CAC number the CMO and Finance can both see move, and it establishes the shared feature store every other agent in the framework depends on.
